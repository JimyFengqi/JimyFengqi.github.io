## 高赞题解
### 解题思路
1. 哈希表存字符出现次数。
2. "balloon" 出现最少的即为结果。

### 代码
```cpp
class Solution {
public:
    int maxNumberOfBalloons(string text) {
        unordered_map<char, int> c;
        for (char s : text) ++c[s];
        return min({c['b'], c['a'], c['l']/2, c['o']/2, c['o']});
    }
};
```
时间复杂度: O(n)
空间复杂度: O(n)
